# 命名规范

## js命名规范：

1. 函数名大驼峰
2. 变量名小驼峰



## css命名规范：

语法：标签类型_标签用途
如：tv_title，用于标题，标签为<text>
--|--|
tv|text|
btn|button|
iv|image|
iv|icon|
